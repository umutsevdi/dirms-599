# Disaster Information And Resource Management System

A web dashboard for monitoring and responding to emergencies using real-time
maps and data.

## Stack

- **Vite** - Build tool and dev server
- **React 19** - UI library
- **TypeScript** - Type checking
- **Material UI** - Component library
- **Leaflet + React-Leaflet** - Maps
- **Recharts** - Charts

## Features

- Interactive map with disaster markers and clustering
- Location search and radius selection tools
- Disaster data tables and detail dialogues
- Charts and statistics
- Geolocation support

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

App runs at `http://localhost:5173`

## Scripts

- `npm run dev` - Start dev server
- `npm run build` - Production build
- `npm run preview` - Preview build
- `npm run lint` - Run ESLint
- `npm run prettier` - Format code
